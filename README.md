# LEGION CNC

Sitio web comercial para LEGION CNC, empresa líder en tecnología CNC industrial en Ecuador y Latinoamérica.

## Descripción

Sitio web de catálogo de productos y servicios para máquinas CNC (Control Numérico Computarizado), incluyendo routers, láseres y plasmas de alta precisión.

## Características

- Catálogo de productos CNC con categorías (Router, Láser, Plasma)
- Buscador de productos
- Carrito de compras con integración WhatsApp
- Formulario de contacto con validación
- Videos de demostración (YouTube)
- Testimonios de clientes
- Presencia en múltiples ciudades de Ecuador
- Panel de administración para gestionar productos y categorías
- Diseño responsivo (móvil y escritorio)
- Modo oscuro/claro
- Animaciones y efectos visuales

## Tecnologías

- HTML5 + CSS3
- JavaScript Vanilla
- LocalStorage para persistencia de datos
- Font Awesome (iconos)
- Google Fonts (Outfit)

## Uso

Simplemente abre `index.html` en un navegador web.

### Admin
Desde el panel de administración puedes:
- Agregar, editar y eliminar productos
- Gestionar categorías
- Ver inventario
